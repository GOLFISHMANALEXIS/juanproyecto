Documentación de mi Aplicación de Cubo Rubik 3D

Descripción de la Aplicación

Esta aplicación simula un cubo Rubik tridimensional (3D) utilizando una matriz 3x3x3. Fue desarrollada con una librería externa encontrada en Internet.

Funcionalidades

  • Mezcla de Caras: La aplicación cuenta con una función para mezclar aleatoriamente todas las caras del cubo.
  • Reinicio: Dispone de un botón para reiniciar el cubo a su estado inicial.

Uso de la Aplicación

Para utilizar la aplicación, simplemente interactúa con los controles proporcionados para mezclar o reiniciar el cubo. Explora las diferentes vistas y posiciones para manipular el cubo virtual.
